# 🌦️ Weather App

A simple and responsive **Weather Application** that allows users to check real-time weather conditions of any city.  
The app fetches live weather data from a **Weather API** and displays important details like temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

- 🌍 Search weather by **city name**
- 🌡️ Displays **current temperature**
- 💧 Shows **humidity levels**
- 🌬️ Displays **wind speed**
- ☁️ Weather condition icons
- 📱 Responsive design
- ⚡ Real-time weather data using API

---

## ⚙️ How It Works

1. User enters a **city name**.
2. JavaScript sends a request to the **Weather API**.
3. The API returns **weather data in JSON format**.
4. The app extracts and displays:
   - Temperature
   - Humidity
   - Wind speed
   - Weather condition
